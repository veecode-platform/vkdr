Documentation coming soon.

## Commands

- `vkdr infra up` - Create and start a local k3d cluster
- `vkdr infra down` - Remove the local k3d cluster
- `vkdr infra start` - Start a stopped cluster
- `vkdr infra stop` - Stop a running cluster
- `vkdr infra expose` - Expose cluster services
- `vkdr infra createtoken` - Create a service account token
- `vkdr infra getca` - Get cluster CA certificate
- `vkdr infra explain` - Show this documentation
